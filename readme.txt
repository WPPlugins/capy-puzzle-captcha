Prevent spams with hassle free captcha.

== Description ==

This plugin uses Capy Puzzle CATCHA for account hijacking prevention. CAPY Puzzle CAPTCHAs use high security technology with a game-like user friendly interface.
As a registered user you will be able to have customize security levels and design appearances for your website. Before using this plugin you will need to register by signing up at http://www.capy.me/.

== Installation ==

1. Upload `capy-wp` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
